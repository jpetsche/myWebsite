Started August 2016

Developing a top-down, real-time 2D web-based multiplayer game. Currently in production. We are using HTML5, CSS3, PHP, Javascript, JSCanvas, Nodejs, Websockets, MySQL, and many other built in JavaScript libraries. Eras also uses an account and chat system that is implemented with MySQL, PHP by means of AJAX, and a Database.
